# Auth Images

## Placeholder untuk images carousel di halaman Login

Simpan 3 gambar dengan nama berikut di folder ini:

1. **auth-slide-1.jpg** - Gambar untuk slide pertama (Team collaboration theme)

    - Recommended size: 1920x1080px
    - Tema: Tim bekerja sama, kolaborasi

2. **auth-slide-2.jpg** - Gambar untuk slide kedua (Security theme)

    - Recommended size: 1920x1080px
    - Tema: Keamanan, enkripsi dokumen

3. **auth-slide-3.jpg** - Gambar untuk slide ketiga (Version control theme)
    - Recommended size: 1920x1080px
    - Tema: Version control, tracking changes

## Cara Pakai

1. Download atau buat 3 gambar sesuai tema di atas
2. Rename sesuai dengan nama file di atas
3. Copy ke folder `public/images/auth/`
4. Refresh halaman login untuk melihat hasilnya!

## Optional: Gunakan Gradient sebagai Background

Jika tidak ada gambar, carousel akan menampilkan gradient background dengan icon FontAwesome (sudah configured).
